# 03_case

This directory contains a complete OpenFOAM HiSA simulation case for the NASA ROBIN rotor–fuselage configuration. The case is ready to run using HiSA on `OpenFOAM-v2512`.

## Case Structure

```
case
├─ 0                   # Initial conditions
├─ constant
│   ├─ dynamicMeshDict # Rotor multibody motion
│   └─ polyMesh        # Copied from 02_mesh/robinmesh/constant/polyMesh
└─ system
    ├─ controlDict
    ├─ decomposeParDict
    ├─ fvSchemes
    └─ fvSolution
```

## Initial Conditions

Initial conditions are defined in: `0/`

Freestream conditions are defined in: `0/include/freeStreamConditions`


## Rotor Motion

Rotor motion is defined in: constant/dynamicMeshDict

The multibody motion solver implementation can be found at 
https://github.com/bosung-gotocloud/gotocfd/tree/main/multiMotionSolver

Additional details are described in the associated paper [Validation_of_OpenFOAM_for_NASA_ROBIN](https://github.com/bosung-gotocloud/nasa-robin-cfd/blob/main/05_paper/Validation_of_OpenFOAM_for_NASA_ROBIN.pdf)

## Parallel Decomposition and AMI Handling

HiSA requires AMI patches to remain on a single processor during domain decomposition. This is handled using faceSets generated by topoSet.

Generate AMI face sets using `topoSet`. AMI face sets correspond to:
```
bladezone000
bladezone090
bladezone180
bladezone270
rotorzone
```

Then decompose the domain for parallel execution `decomposePar`

Number of subdomains can be adjusted in `system/decomposeParDict`.

## Numerical Schemes

### Time Integration

Dual-time stepping defined in `system/fvSchemes` is used:
```
ddtSchemes
{
    default bounded dualTime rPseudoDeltaT backward;
}
```

### Linear Solver

Defined in `system/fvSolution` as
```
Flow solver: GMRES
Preconditioning: LUSGS

Pseudo-time iteration parameters:
nPseudoCorr     10
nPseudoCorrMin  8
pseudoTol       1e-3

Local time stepping is enabled.
```

## Simulation Control

Defined in `system/controlDict` as :

```
Simulation parameters:
Total revolutions      : 10
Time per revolution    : 0.02972 s
Steps per revolution   : 200
Total simulation time  : 0.2972 s
deltaT = 0.0001486
```

Output is written every 10 time steps.

## Probe Locations

Pressure probes are defined using pointProbes in `system/controlDict`. Probe locations correspond to experimental measurement points used for validation.

## Running the Simulation

After copying the mesh from `02_mesh/robinmesh/constant/polyMesh` into `case/constant/` and generating AMI face sets:

```
cd case
topoSet
decomposePar -force
mpirun -np 64 hisa -parallel
```

Adjust the number of cores (-np) as needed.

## Post-Processing

