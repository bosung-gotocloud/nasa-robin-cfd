# NASA ROBIN Rotor CFD Case (HiSA / OpenFOAM)

Reproducible CFD dataset for the NASA ROBIN rotor–fuselage configuration using OpenFOAM and HiSA.

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.XXXXXXX)

This repository provides a reproducible CFD simulation case for the NASA ROBIN helicopter configuration using OpenFOAM with the HiSA solver.

The repository includes:

- geometry files
- mesh files
- a complete OpenFOAM/HiSA simulation case
- post-processing data and visualization files
- experimental reference data for validation

The dataset allows reproduction of the **ROBIN rotor–fuselage interaction simulation** and comparison with experimental measurements reported in NASA technical reports.

# Workflow

The CFD workflow follows these steps:

```
Geometry → Mesh → CFD Case → Simulation → Post-processing
```

1. Geometry definition (`01_geometry`)
2. Mesh generation (`02_mesh`)
3. OpenFOAM / HiSA simulation case (`03_case`)
4. Simulation execution
5. Post-processing and validation (`04_post`)

---

# 1 Geometry

Directory:

```
01_geometry
```

Contains STEP files for the **ROBIN fuselage, pylon, and rotor blade geometry**.

The rotor blade geometry is based on a **NACA0012 airfoil with −8° twist and rotor radius 0.86R**.

The fuselage and pylon geometry were generated using:

```
https://github.com/bosung-gotocloud/gotocfd/tree/main/stepROBIN
```

---

# 2 Mesh

Directory:

```
02_mesh
```

The mesh is generated using **SALOME** and exported to OpenFOAM format.

After mesh generation, copy the mesh into the simulation case:

```bash
cp -r 02_mesh/robinmesh/constant/polyMesh 03_case/case/constant/
```

# 3 Simulation Case (HiSA)

Directory:

```
03_case/case
```

This directory contains a **complete OpenFOAM case** that can be directly executed using **HiSA v0.151**.

Structure:

```
case
├─ 0
├─ constant
│   ├─ dynamicMeshDict
│   └─ polyMesh
└─ system
```

# Parallel Decomposition and AMI Handling

HiSA requires AMI patches to remain on a **single processor** during domain decomposition.

This is handled using **faceSets generated by topoSet**.

---

## topoSet

AMI face sets are generated using:

```bash
topoSet
```

These face sets correspond to:

```
bladezone000
bladezone090
bladezone180
bladezone270
rotorzone
```

---

## decomposePar

Parallel decomposition uses the **scotch** method.

Example configuration:

```
numberOfSubdomains 64;
```

The number of subdomains can be changed depending on the number of available CPU cores.

AMI interfaces are handled using:

```
singleProcessorFaceSets
```

---

# Numerical Schemes

## Time Integration

Defined in:

```
system/fvSchemes
```

Dual-time stepping is used:

```
ddtSchemes
{
    default bounded dualTime rPseudoDeltaT backward;
}
```

---

## Linear Solver

Defined in:

```
system/fvSolution
```

The flow solver uses:

```
GMRES
```

with

```
LUSGS preconditioning
```

Pseudo-time iteration parameters include:

```
nPseudoCorr     10
nPseudoCorrMin  8
pseudoTol       1e-3
```

Local time stepping is enabled.

---

# Simulation Control

Defined in:

```
system/controlDict
```

Simulation parameters:

```
Total revolutions      : 10
Time per revolution    : 0.02972 s
Steps per revolution   : 200
Total simulation time  : 0.2972 s
```

Time step:

```
deltaT = 0.0001486
```

Solution output is written:

```
every 10 time steps
```

---

# Probe Locations

Pressure probes are defined using:

```
pointProbes
```

in `controlDict`.

These probes correspond to measurement locations used for comparison with experimental data.

Recorded fields include:

```
p
U
rho
rhoE
```

# Running the Simulation

After copying the mesh:

```bash
cd 03_case/case
```

Generate AMI face sets:

```bash
topoSet
```

Decompose the domain:

```bash
decomposePar
```

Run the simulation:

```bash
mpirun -np 64 hisa -parallel
```

# Post-Processing

Directory:

```
04_post
```

This directory contains:

- experimental reference data
- processed CFD results
- ParaView state files

Experimental data are taken from:

Mineck, R. E., and Gorton, S. A.,  
*Steady and Periodic Pressure Measurements on a Generic Helicopter Fuselage Model in the Presence of a Rotor*,  
NASA/TM-2000-210286, 2000.

These include **Table 21 and Table 22** pressure measurements used for validation.
